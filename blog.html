Quesion-1: What are the defferent ways to select an element in the DOM.

There are several ways to select an element in the DOM. With id, with className, tag name etc.

Quesion-2: What is the difference between innerHTML, innerText, and textContent ?

With innerHTML we can get all the html tag attributes even spaces. With innerText we can have only texts inside of a tag.Excluding html tags name spaces etc

Quesion-3: What is event delegation in the DOM?

Event delegation is where the event get handled by its parents element instead of directly on the current element.

Quesion-4:What is event bubbling in the DOM?

Event bubbling is where event start from current element to its top most parent element just like bubbling water. like current tag element to document which is the top ancestor.

Quesion-5: How do you create, add, and remove elements using JavaScript?

Firstly we have to create the element with document.createElement(). Then add all the text and html inside of it. And then attach the newly created element to the the dom with .append method. We can also attach with .nodeChild method which will work as a child element.
